Download

先安装两个工具

请根据自己的系统下载 Codex、Claude Code 或 CC Switch。安装完成后再继续配置。

C

Codex

用于本地 AI 编程协作。下载后可通过 CC Switch 本地代理接入 API 服务。

打开 Codex 下载页 备用入口：https://openai.com/codex/
S

CC Switch

用于管理 Codex 和 Claude Code 的本地代理、供应商配置与线路切换。

打开 CC Switch 下载页 官网入口：https://ccswitch.ai/

Setup

配置步骤

优先使用 API 页面的一键导入。只有导入失败或需要自定义时，再使用手动配置。

Recommended

推荐：在 API 页面点击“导入到 CCS”

打开 API Keys 页面，找到要使用的 Key，点击“导入到 CCS”。随后选择 Codex 或 Claude Code，CC Switch 会自动接收配置。两个客户端的流程一致。

API 页面 导入到 CCS 选择客户端 打开工具测试
1

打开 API 页面

进入 API Keys 页面，确认当前账号已经有可用的 API Key。

客户端：Codex 或 Claude Code
2

点击导入到 CCS

Claude Code：选择 Claude Code
两者步骤相同

应用类型：Codex 或 Claude Code
状态：已保存

Troubleshooting

常见问题排查

无法连接

优先重新在 API 页面点击“导入到 CCS”，并确认 CC Switch 已经打开。

提示 401 或 403

通常是 API Key 错误、过期，或当前账号没有对应模型权限。

提示 404

通常是 API 地址错误。请确认手动配置时请求地址填写为 https://api.mecho.top，末尾不要加斜杠。

配置后仍失败

请重启客户端，并确认 CC Switch 当前启用的是正确配置。
